## Quy ước chung

| Ký hiệu | Ý nghĩa |
|---|---|
| `*` sau tên field | Trường bắt buộc |
| `?` sau tên field | Trường tùy chọn |
| `verifyToken` | Kiểm tra JWT hợp lệ trong header `Authorization: Bearer <token>` |
| `checkRole([...])` | Kiểm tra `vai_tro.ma_vai_tro` có nằm trong danh sách cho phép không |
| `checkTicketAccess` | Requester chỉ truy cập ticket của mình; L1/L2 truy cập ticket nhóm mình |
| `checkTicketAssignment` | L1/L2 chỉ thao tác trên ticket đang được phân công cho mình |
| `filterByRole` | Tự động thêm điều kiện WHERE vào query dựa theo vai trò người dùng |
| `validateBody(schema)` | Validate request body bằng Zod schema tương ứng |
